opensubtitleHashFile(name)

source code 

Hashes a file using OpenSubtitle's method.

> In natural language it calculates: size + 64bit chksum of the first and > last 64k (even if they overlap because the file is smaller than 128k).

A slightly more Pythonic version of the Python solution on.. http://trac.opensubtitles.org/projects/opensubtitles/wiki/HashSourceCodes
